# Tree-sitter Neruda Grammar

This is the Tree-sitter grammar definition for the Neruda language, designed to provide syntax highlighting and parsing support for text editors.

## Features

- Syntax highlighting for Neruda language
- Support for all Neruda language constructs including:
  - Functions and systems
  - Schedulers and components
  - Variables and types
  - Control flow statements
  - Expressions and literals
- Proper handling of comments and documentation strings

## Installation

To use this grammar in a text editor that supports Tree-sitter:

1. Install the Tree-sitter CLI: `npm install -g tree-sitter-cli`
2. Clone or install this repository
3. Build the grammar: `tree-sitter generate`

## Usage

This grammar can be used to provide:
- Syntax highlighting in text editors
- Code folding
- Language server support
- Structural editing capabilities
